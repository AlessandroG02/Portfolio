GeoRoyale

GeoRoyale è un'applicazione web per quiz geografici sviluppata con Spring Boot 3.5 e Java 17.

Funzionalità principali

  • 5 tipologie di quiz: Capitali, Nazioni, Bandiere, Continenti, Higher or Lower (popolazione)
  • Due modalità di gioco: allenamento gratuito e quiz autenticati con punteggi
  • Sistema di sessioni: 10 domande consecutive con scoring progressivo
  • Profilo utente: statistiche, livelli, cronologia quiz e record personali

Reale ITES – Sito Web

Il "Simulatore Polizze Reale ITES" è un'applicazione web che permette agli utenti di calcolare preventivi assicurativi per polizze Auto, Casa e Vita. Sviluppata con HTML, CSS e JavaScript vanilla, offre funzionalità di:

  • Login utente
  • Simulazione con parametri personalizzabili
  • Storico delle simulazioni
  • Confronto tra preventivi
  • Pagine informative aziendali e riconoscimenti

L'interfaccia è responsiva con un design coerente in blu. I dati vengono memorizzati localmente nel browser dell’utente. L'autenticazione client-side limita l’uso dell’app a scopi dimostrativi, non per produzione.

Learning by Project

Registro elettronico digitale per gestire le presenze di studenti e tirocinanti (chiamati "recruit") sotto la supervisione di tutor aziendali, sostituendo il tradizionale registro cartaceo. Inizialmente sviluppato con JPA Repository, Hibernate e Lombok, è stato successivamente aggiornato utilizzando JDBC e migliorando il frontend con Angular.
