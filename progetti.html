GeoRoyale

Geo Royale è un'applicazione web per quiz geografici sviluppata con Spring Boot 3.5 e Java 17. Funzionalità Principali: 5 tipologie di quiz: Capitali, Nazioni, Bandiere, Continenti, Higher or Lower (popolazione) Due modalità: Allenamento gratuito vs Quiz autenticati con punteggi Sistema sessioni: 10 domande consecutive con scoring progressivo Profilo utente: Statistiche, livelli, cronologia quiz e record personali

Reale ITES – Sito Web

Il "Simulatore Polizze Reale ITES" è un'applicazione web che permette agli utenti di calcolare preventivi assicurativi per polizze Auto, Casa e Vita. Sviluppata con HTML, CSS e JavaScript vanilla, offre funzionalità di login, simulazione con parametri personalizzabili, storico delle simulazioni e confronto tra preventivi. L'interfaccia è responsiva con un design coerente in blu. Include anche pagine informative come riconoscimenti aziendali. L'applicazione memorizza i dati localmente nel browser dell'utente. Sebbene funzionale, l'autenticazione client-side ne limita l'uso a scopi dimostrativi piuttosto che in produzione.
