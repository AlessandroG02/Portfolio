Georoyale

Videogioco multiplayer competitivo basato su conoscenze geografiche e velocità di risposta

Java Socket TCP MySQL API Geografiche

Descrizione del Progetto

Geo Royale è un'applicazione web per quiz geografici sviluppata con Spring Boot 3.5 e Java 17.

Funzionalità Principali

  • 5 tipologie di quiz: Capitali, Nazioni, Bandiere, Continenti, Higher or Lower (popolazione)
  • Due modalità: Allenamento gratuito e Quiz autenticati con punteggi
  • Sessioni di 10 domande con scoring progressivo
  • Profilo utente con statistiche, livelli, cronologia e record personali

Architettura

  • Backend: Spring Boot con Spring Security, JPA/Hibernate, database H2
  • Frontend: Thymeleaf con CSS responsive e design moderno

Caratteristiche

  • 168 paesi con dati reali (popolazione, capitali, regioni)
  • Sistema punteggi persistente con tracking di accuratezza e streak
  • Interfaccia responsive con stile gaming e gradients verdi
  • Ruoli differenziati: utenti anonimi per allenamento, autenticati per competizione

Screenshots del Progetto

Schermata di Accesso

Interfaccia per effettuare il login con le proprie credenziali prima di accedere alle modalità di gioco.

Registrazione Nuovo Utente

Modulo per creare un nuovo account inserendo nome utente, email e password.

Pagina iniziale

Schermata principale del gioco con opzioni per accedere alle diverse modalità di gioco e visualizzare le statistiche. Include pulsanti per iniziare una nuova partita, visualizzare il profilo e accedere alla classifica globale.

Modalità Allenamento

Sessione di gioco in singolo per esercitarsi e migliorare le proprie abilità geografiche.

Modalità "Guess the Capital"

Il giocatore deve indovinare la capitale corretta tra le opzioni disponibili nel minor tempo possibile.

Modalità "Higher or Lower"

Sfida a tempo in cui bisogna identificare rapidamente località geografiche casuali sulla mappa.

Profilo Utente

Pagina personale con statistiche di gioco, punteggi raggiunti e impostazioni del profilo.

Modalità Quiz

Rispondi a domande geografiche a scelta multipla per accumulare punti e scalare la classifica.

Tecnologie Utilizzate

  • Frontend: Thymeleaf templates con CSS responsive e design moderno.
  • Backend: Spring Boot 3.5 con Spring Security, JPA/Hibernate per la gestione dati e sicurezza.
  • Database: Database in-memory H2 per sviluppo e test, con possibilità di switch a database relazionali.
  • Sicurezza: Autenticazione completa, password criptate e protezione degli endpoint.
  • Altro: Design responsive con interfaccia gaming-style e gestione sessioni di quiz.
