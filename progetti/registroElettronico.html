Registro Elettronico

Sistema gestionale completo per la gestione di tutor e tutorati con dashboard avanzata e tracking presenze

Spring Boot Hibernate Lombok HTML5 CSS3 JavaScript H2 Database

Descrizione del Progetto

Registro Elettronico è un sistema gestionale completo sviluppato per la gestione efficiente di tutor e tutorati, con funzionalità avanzate di tracking delle presenze e dashboard di controllo.

Funzionalità Principali

  • Dashboard Completa: Panoramica con statistiche tutor attivi, tutorati registrati, presenze totali e presenze da confermare
  • Gestione Tutor: Sistema completo per registrazione, modifica ed eliminazione dei tutor con dati anagrafici completi
  • Gestione Tutorati: Registrazione e gestione degli studenti tutorati con assegnazione ai tutor
  • Sistema Presenze: Tracking automatico delle presenze con stati (Presente, Assente Giustificato, Assente Non Giustificato)
  • Conferma Presenze: Sistema di validazione delle presenze con possibilità di conferma o modifica
  • Storico Completo: Mantenimento dello storico di tutte le attività e presenze registrate

Caratteristiche Tecniche

  • Architettura MVC: Sviluppato con Spring Boot seguendo il pattern Model-View-Controller
  • ORM Avanzato: Utilizzo di Hibernate per la gestione del database e delle relazioni
  • Code Clean: Implementazione di Lombok per ridurre il boilerplate code
  • Database Embedded: H2 Database per sviluppo rapido e testing
  • Interface Responsive: Design ottimizzato per tutti i dispositivi
  • API RESTful: Endpoints ben strutturati per tutte le operazioni CRUD

Obiettivi del Progetto

  • Digitalizzare completamente il processo di gestione tutor-tutorati
  • Automatizzare il tracking delle presenze riducendo errori manuali
  • Fornire dashboard intuitive per monitoraggio e reporting
  • Creare un sistema scalabile e facilmente manutenibile
  • Implementare un flusso di lavoro efficiente per la conferma delle presenze

Screenshots del Progetto

Home - Gestione Presenze

Sistema avanzato per il tracking delle presenze con interfaccia intuitiva. Mostra le presenze registrate con data, stato (Presente, Assente Giustificato, Assente Non Giustificato) e possibilità di conferma. Include funzionalità per aggiungere nuove presenze e modificare quelle esistenti, garantendo un controllo completo del registro.

Gestione Tutorati

Interfaccia completa per la gestione dei tutorati registrati nel sistema. Include la visualizzazione di tutti gli studenti con informazioni dettagliate come nome completo, email, tutor assegnato e numero di assenze non giustificate. Permette l'aggiunta di nuovi tutorati tramite form dedicato e la gestione delle relazioni tutor-studente.

Form Registrazione Tutor

Form completo per la registrazione di nuovi tutor nel sistema. Include tutti i campi necessari come nome, cognome, email, telefono e data di nascita con validazione lato client e server. Design pulito e user-friendly con controlli di input ottimizzati per una compilazione rapida e senza errori.

Dashboard di Controllo

Dashboard principale del sistema che fornisce una panoramica completa delle attività. Mostra statistiche in tempo reale come numero di tutor attivi, tutorati registrati, presenze totali e presenze in attesa di conferma. Include sezioni dedicate per presenze da confermare con azioni rapide di validazione e modifica.

Tecnologie Utilizzate

  • Backend: Spring Boot framework con architettura MVC, Hibernate ORM per la persistenza dei dati
  • Database: H2 Database embedded per sviluppo e testing rapido
  • Frontend: HTML5, CSS3 responsive e JavaScript per interazioni dinamiche
  • Strumenti: Lombok per riduzione boilerplate code, Maven per dependency management
  • Sicurezza: Validation e controlli di integrità dati
